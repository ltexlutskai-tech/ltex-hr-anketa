# Інструкція з налаштування: Анкета L-TEX → Google Sheets + Telegram

## Крок 1: Створити Google Sheet

1. Відкрийте https://sheets.google.com
2. Створіть нову таблицю з назвою **"L-TEX Кандидати"**
3. Залиште першу вкладку порожньою — скрипт створить заголовки автоматично

## Крок 2: Додати Google Apps Script

1. У Google Sheet відкрийте **Розширення → Apps Script**
2. Видаліть весь код що є за замовчуванням
3. Скопіюйте весь вміст файлу `google_apps_script.js` та вставте
4. Натисніть **Ctrl+S** (зберегти)

## Крок 3: Створити Telegram бота

1. Відкрийте Telegram, знайдіть **@BotFather**
2. Надішліть: `/newbot`
3. Введіть ім'я бота, наприклад: `L-TEX HR Bot`
4. Введіть username, наприклад: `ltex_hr_bot`
5. BotFather надасть вам **токен** — скопіюйте його
6. Вставте токен у `google_apps_script.js` → рядок `BOT_TOKEN`

### Отримати Chat ID:

**Варіант А** — особистий чат:
1. Напишіть своєму боту будь-яке повідомлення
2. Відкрийте: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
3. Знайдіть `"chat":{"id": ЧИСЛО}` — це ваш Chat ID

**Варіант Б** — група:
1. Додайте бота у групу
2. Напишіть повідомлення в групі
3. Відкрийте: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
4. Chat ID групи буде від'ємним числом (наприклад: `-1001234567890`)

Вставте Chat ID у `google_apps_script.js` → рядок `CHAT_ID`

## Крок 4: Опублікувати Apps Script як Web App

1. В Apps Script натисніть **Розгортання → Нове розгортання**
2. Тип: **Веб-додаток**
3. Опис: `L-TEX Anketa v1`
4. Виконувати як: **Я (ваш email)**
5. Хто має доступ: **Будь-хто**
6. Натисніть **Розгорнути**
7. Надайте дозволи (Google попросить підтвердити)
8. Скопіюйте **URL веб-додатку**

## Крок 5: Підключити HTML-форму

Відкрийте файл `anketa_menedzher_prodazhu.html` та знайдіть блок CONFIG:

```javascript
const CONFIG = {
  GOOGLE_SCRIPT_URL: 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE',
  SEND_TO_SHEETS: false
};
```

Замініть на:

```javascript
const CONFIG = {
  GOOGLE_SCRIPT_URL: 'https://script.google.com/macros/s/ВАШ_ID/exec',
  SEND_TO_SHEETS: true
};
```

## Крок 6: Тестування

1. Відкрийте Apps Script
2. Виберіть функцію `testSetup` зі списку
3. Натисніть **Виконати (Run)**
4. Перевірте:
   - У Google Sheet з'явилась вкладка "Кандидати" з тестовим рядком
   - У Telegram прийшло повідомлення з даними тестового кандидата
5. Відкрийте HTML-анкету, заповніть і відправте
6. Перевірте що дані з'явились у Sheet + Telegram

## Оновлення Apps Script

Після зміни коду:
1. Apps Script → **Розгортання → Керувати розгортаннями**
2. Натисніть олівець (редагувати)
3. Версія: **Нове розгортання**
4. Натисніть **Розгорнути**

URL залишається незмінним.

## Безпека

- BOT_TOKEN зберігається тільки в Google Apps Script (серверна сторона)
- В HTML-файлі токен НЕ видно
- Google Sheet доступний тільки вам (автору)
- Apps Script працює від вашого Google акаунту
