# HR-LTEX Skill — Human Resources for L-TEX

## Company Context
- **Company**: L-TEX (brands: Secondopt, Bricabrac)
- **Business**: Wholesale second-hand & Bric-a-Brac, Lutsk Ukraine
- **Sites**: secondopt.com.ua, bricabrac.com.ua
- **Experience**: 9 years, 3000+ clients
- **Supply**: UK, Poland, Canada, Germany, Italy
- **Products**: Clothing, shoes, bags, home goods, toys, Bric-a-Brac, cosmetics
- **Grades**: Extra, Cream, 1st grade, 2nd grade, Stock
- **Prices**: EUR, minimum order from 10kg
- **Channels**: Telegram (@L_TEX), Viber, phone, YouTube
- **Location**: Piddubci, Lutsk district, Volyn region

## Key Positions
1. **Sales Manager** — wholesale B2B sales, client base 3000+, EUR pricing, Telegram/Viber
2. **Warehouse Worker** — sorting, grading (Extra/Cream/1st/2nd), packing, inventory
3. **Office Manager** — admin, documentation, coordination

## Document Types

### 1. Posadova Instruktsiia (Job Description)
Structure:
1. Загальні положення (General provisions)
2. Підпорядкування (Reporting structure)
3. Посадові обов'язки (Duties — 15-20 specific points)
4. Права (Rights)
5. Відповідальність (Responsibility)
6. KPI та показники (KPI metrics — quantitative)
7. Вимоги до кандидата (Requirements)
8. Умови праці (Work conditions)
9. Підписи (Signatures: Director + Employee + Date)

Language: Ukrainian, formal style
Format: .docx via docx skill

### 2. Candidate Questionnaire (Anketa)
Format: Single HTML file, mobile-first
Structure: 5 steps with progress bar
- Step 1: Personal data
- Step 2: Work experience
- Step 3: Skills relevant to L-TEX
- Step 4: Situational questions (3 cases specific to wholesale secondhand)
- Step 5: Motivation & expectations
End: Thank you page with L-TEX contacts

### 3. KPI System
For Sales Manager:
- Monthly revenue plan (EUR)
- New clients per month
- Client retention rate
- Average order value
- Response time to inquiries

### 4. Sales Scripts
Tone: Friendly, knowledgeable, helpful
Context: B2B wholesale, clients are entrepreneurs
Key topics: grades explained, pricing in EUR, delivery options

## Sales Manager Specifics
- Works with existing 3000+ client base
- Attracts new wholesale buyers (shop owners, market traders)
- Sends video reviews from YouTube channel
- Explains grade system (Extra/Cream/1st/2nd/Stock)
- Handles EUR pricing + currency exchange questions
- Communicates via Telegram, Viber, phone

## Output Rules
- Always Ukrainian language for documents
- Official/formal tone for job descriptions
- Use docx skill for .docx output
- Use frontend-design skill for HTML output
